# Laboratorios 3 y 4 - Microcontroladores ELEB8092

Sistema de vision artificial con Raspberry Pi 4 y ESP32-CAM.

## Contenido

- `esp32_cam_stream/`: firmware para ESP32-CAM AI-Thinker con stream MJPEG.
- `rasberryesp32.py`: servidor Raspberry Pi para consumir el stream y ejecutar deteccion.
- `lab4_scripts/`: pruebas de entorno, captura local, deteccion y ANPR.
- `informe3/`: informe integrado en LaTeX, PDF, imagenes y codigos de entrega.
- `yolov8n.onnx`: modelo YOLOv8 Nano exportado para OpenCV DNN.
